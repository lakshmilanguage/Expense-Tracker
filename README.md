# Expense Tracker (Python)
A simple command-line Expense Tracker built using Python. This application helps users record, view, calculate, and delete daily expenses.

Features
Add new expenses
View all expenses
Calculate total expenses
Delete expenses
User-friendly menu-driven interface

Technologies Used
Python 3
How It Works
Menu Options

Add Expense
Enter expense category
Enter expense amount
Expense is saved in the list

View Expenses
Displays all recorded expenses with serial numbers

Total Expense
Calculates and displays the total amount spent

Delete Expense
Removes an expense using its serial number

Exit
Closes the application
